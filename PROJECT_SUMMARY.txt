🎯 NASIL KULLANILIR?
════════════════════════════════════════════════════════════════════

▶️ YÖNTEM 1: Web Arayüzü (Önerilen)
─────────────────────────────────
1. Terminal açın ve MVC başlatın:
   cd Pos.Web
   dotnet run --urls "https://localhost:7002"

2. Tarayıcıda açın: https://localhost:7002
3. Giriş yapın: admin@pos.com / Admin123
4. Dashboard'dan modüllere erişin

▶️ YÖNTEM 2: API Üzerinden
─────────────────────────────────
1. Swagger UI: https://localhost:7001/swagger
2. Authorize butonuna tıklayın
3. Login endpoint'i çalıştırın → Token alın
4. Token'ı "Bearer {token}" formatında girin
5. Diğer endpoint'leri test edin

▶️ YÖNTEM 3: cURL ile
─────────────────────────────────
# Token al
TOKEN=$(curl -s -k -X POST https://localhost:7001/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email": "admin@pos.com", "password": "Admin123"}' \
  | python3 -c "import sys, json; print(json.load(sys.stdin)['token'])")

# Ürünleri listele
curl -k -H "Authorization: Bearer $TOKEN" \
  https://localhost:7001/api/products

# Sipariş oluştur
curl -k -X POST https://localhost:7001/api/orders \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "customerId": "CUSTOMER_GUID",
    "notes": "Test siparişi",
    "items": [
      {"productId": "PRODUCT_GUID", "quantity": 2}
    ]
  }'
